### Selected Work

**Yassir Distribution Management System** — *Originova*
Distribution & inventory platform for up to 1,000 concurrent users across 5 operational roles. 150+ table PostgreSQL schema, 500+ REST endpoints, and complex financial workflows (ledgers, debt tracking, reconciliation) with async pipelines on Celery/Redis. Deployed to DigitalOcean via Docker, Nginx, Gunicorn & GitLab CI/CD.

**Lift & Cable Installation Management Platform** — *Freelance*
End-to-end enterprise system: project lifecycle, inventory with reorder alerts & margin tracking, multi-line invoicing, and a scheduling calendar — secured with JWT + role-based access control. Built with DRF, React, TypeScript & Tailwind.

**Official State Library of Tisemsilt**
Full-stack platform serving thousands of monthly users, integrated with the PMB library system over JRPC. Cut response times ~40% with Redis caching. Django · PostgreSQL · Redis · Docker.

**Lotok — Car Rental Platform**
Backend services for auth, booking & fleet management, plus a native Android client in Kotlin & Jetpack Compose with Google / Facebook sign-in.
